This project is a simple and interactive Expense Tracker web application built using HTML, CSS, and JavaScript. It allows users to record financial transactions by selecting the date, transaction type (Expense or Savings), and amount. The application dynamically calculates and displays total income, total savings, and total expenses based on the entered data.

Functionality:
Users can select a transaction date, choose the transaction type, and enter the transaction amount. Upon adding a transaction, the corresponding values are automatically updated and reflected in the summary section. All transactions are incrementally added to structured tables for better tracking and review.

Key Features:
Date-based transaction entry
Transaction type selection (Expense, Savings)
Real-time calculation of income, savings, and expenses
Automatic data updates without page reload
Transaction history displayed in tabular format
Clean and responsive user interface

Technologies Used:
HTML for structure
CSS for styling and layout
JavaScript for logic, calculations, and DOM manipulation

Purpose:
This project is designed to strengthen fundamental web development skills and demonstrate the use of JavaScript for real-time data handling and UI updates.
